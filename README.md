# 🛒 Amazon Clone

This is a simple **Amazon Clone** project built using **HTML and CSS**.
It replicates the basic structure and design of the Amazon homepage.

---

## 🚀 Features

* 🏠 Homepage layout similar to Amazon
* 🖼️ Hero section with banner image
* 📦 Product boxes with images
* 🎨 Clean and responsive UI
* 🧭 Navigation bar with logo and menu

---

## 🛠️ Tech Stack

* HTML5
* CSS3

---

## 📂 Project Structure

```
Amazon-Clone/
│── index.html
│── style.css
│── hero_image.jpg
│── amazon_logo.png
│── box1_image.jpg
│── box2_image.jpg
│── ...
```

## ⚡ How to Run

1. Clone the repository:

   ```
   git clone https://github.com/BrijeshNishad-dotcom/Amazon-Clone.git
   ```

2. Open the folder:

   ```
   cd Amazon-Clone
   ```

3. Run the project:

   * Open `index.html` in your browser
